
# setting

```
$ gem install rails -v 4.2.3
```

```
$ rails _4.2.3_ new ./ -d postgresql
```

# make controller

```
$ rails g controller {name(s)} {method}
```

# make model

```
$ rails g model {name}
```

# git 

- ステージに配置していないものを元に戻す

```
$ git reset --hard HEAD
```

- 追跡されていないファイル、ディレクトリをワークツリーから削除する。
```
$ git clean -fd
```
